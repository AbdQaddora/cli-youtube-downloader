
## HOW TO INSTALL

```
npm i abdyts -g
```

## HOW TO USE

```
abdyts 'youtube video link' /*optional flag*/
```

## FLAGS 
***Note:** all flags added after the link except help flag*

**Help command** ```--help```

**To download audio only add flag** ```--audio```

**To customize the video quality add one of the following flags:**

```--hd1080``` -> 1080p

```--hd720``` -> 720p

```--l480``` -> 480p

```--m360``` -> 360p

```--s240``` -> 240p

```--t144``` -> 144p
