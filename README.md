# NODE JS CLI YOUTUBE DOWNLOADER

## HOW TO SETUP

clone the project then open it in the terminal

```
npm i -g
```

## HOW TO USE

```
abdyts 'youtube video link'
```

## FLAGS
**Help command** ```--help```

**To download audio only add flag** ```--audio```

**To customize the video quality add one of the following flags:**

```--hd1080``` -> 1080p

```--hd720``` -> 720p

```--l480``` -> 480p

```--m360``` -> 360p

```--s240``` -> 240p

```--t144``` -> 144p
